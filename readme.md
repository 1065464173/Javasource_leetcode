✅ 《剑指OfferI》/ 栈与队列（困难）

#### 2021/12/7

回溯算法就是纯暴力穷举，复杂度一般都很高.回溯的套路（考虑三个问题）：

1. 路径：也就是已经做出的选择。
2. 选择列表：也就是你当前可以做的选择。
3. 结束条件：也就是到达决策树底层，无法再做选择的条件。

```
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return

    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
```
